Breeze Admin

Introdução

O Breeze Admin é um pacote para Laravel que adiciona um painel administrativo baseado no Breeze, oferecendo rotas, views, autenticação e módulos de administração prontos para uso.

Ideal para projetos que precisam de um painel rápido, modular e simples.

Instalação

Instale o pacote via Composer:

composer require kazuha4-sys/breeze-admin

Depois publique os arquivos:

php artisan vendor:publish --tag=breeze-admin

Configuração

Após publicar, o pacote cria rotas, controllers e views prontas. Certifique-se de que sua autenticação Breeze esteja instalada.

Gerenciamento de Usuários

O painel fornece:

  • Listagem paginada
  • Edição
  • Criação
  • Banimento
  • Reset de senha

Permissões & Roles

O pacote oferece integração com roles e permissões simples e funcionais.

Views & Layouts

As views usam Tailwind e seguem o estilo do Breeze original, mantendo leveza e minimalismo.

Rotas

Route::prefix('admin')->group(function() {
    require __DIR__.'/admin.php';
});

Migrations

O pacote adiciona migrations para tabelas administrativas quando necessário.

Componentes do Painel

Inclui tabela dinâmica, search bar, filtros e componentes blade reutilizáveis.
